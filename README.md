## General ##
Designed and implemented a robust CRUD (Create, Read, Update, Delete) API for a Netflix-like website using Node.js, Nest.js, and a NoSQL database. 

- **Database** : MongoDb
- **Framework** : Node.js, Nest.js
- **Focus** : CRUD API endpoints development
